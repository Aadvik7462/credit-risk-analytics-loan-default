# Credit Risk Analytics & Loan Default Prediction

## Project Overview

This project focuses on analyzing loan applicant data and predicting loan approval risk using Machine Learning and Power BI.

The solution helps financial institutions identify high-risk customers, evaluate loan default probability, analyze customer segments, and support data-driven lending decisions.

---

## Business Objectives

* Identify customers with high default risk
* Analyze factors influencing loan approval
* Segment customers based on risk profiles
* Calculate money at risk
* Visualize loan performance through interactive dashboards
